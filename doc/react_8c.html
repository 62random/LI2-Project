Detailed Description

Este ficheiro contém o conjunto de funções que são chamadas ao aplicar ao estado do jogo o movimento que o jogador pretende efetuar. Ou seja, efetua-se o movimento do jogador, e de seguida chama-se a função react (e as outras funções deste módulo, consequentemente).
